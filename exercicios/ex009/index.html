Trabalhando com listas

Listas ordenadas:

São aquelas em que a ordem importa, tendo que seguir um padrão. Para utilizá-las é necessário a tag <ol>. É possível personalizar sua lista com o atributo type (que indica se sua lista seguirá padrão de números, letras ou algarismos romanos, nestes 02 últimos casos, podendo estarem maiúsculas ou não). Então, para adicionar o texto, basta adicionar a tag <li>, antes do que deseja inserir na lista:

  1. Acordar
  2. Tomar banho
  3. Pegar o onibus
  4. Trabalhar
  5. Voltar para casa
  6. Tomar outro banho
  7. Estudar
  8. Ir dormir

Obs:Vale lembrar que o atributo start, em <ol>, indica qual será a posição inicial da sua lista

Listas não ordenadas

Mesmo princípio da lista ordenada, a diferença é que a ordem não importa, tendo, também, o atributo type, que edita como será exibído (disc, circle, square). Para usá-la, basta utilizar a tag <ul>:

  • Adicionar gastos de materiais de limpeza na planilha
  • Levar gaiola de manta sem etiqueta no acabamento
  • Fazer controle semanal de ocorrêrncias

Obs:por algum motivo, o atributo type não se autocompletou... Porem, funcionou sem problemas

Lista de Definições

Não foge muito do que o nome já diz, seguindo os mesmos princípios das listas acima. Para definí-la utilizamos a tag <dl>. Dentro da tag, definimos o termo (<dt>) e sua descrição (<dd>). Esse divisão entre palavra e significado auxilia os mecanismos de busca.

HTML
Uma linguagem de marcação que tem como foco gerar o conteúdo de uma página web
CSS
Uma linguagem de marcação que tem como objetivo estilizar uma página web
JavaScript
Uma linguagem de programação, que pode ser utilizada para adicionar interatividade a página, como animações, menus, etc.
