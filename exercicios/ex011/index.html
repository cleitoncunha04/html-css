Mídias em HTML5:

Imagem dinâmica:

Uma imagem com dimensões variáveis é muito útil quando nos tratamos de responsividade, uma vez que, permite uma maior adaptação do site em diferentes plataformas (smartphone, tablet e PC, por exemplo). Para inserí-las, regatamos a tag <img>, porem, desta vez a envelopamos com a tag <picture>. Dentro desta tag, também adicionamos outra (<source>), que permite definir qual dimensão máxima/mínima (neste caso, a largura) para exibir uma imagem, graças a seu atributo media, alem disso, também é passado como "parâmetro" o caminho da imagem (srcset) e o tipo (type) dela.

Aumente e diminua o tamanho para perceber.

Adicionando audios:

Para inserir aúdios em HTML5 é bem simples, basta envelopar a tag <source> (vista acima), com outra tag: <audio>. Nela devemos utilizar alguns atributos, sendo o primeiro deles o preload, que possui 03 opções:

  • auto, que carrega o audio ao abrir o site, mesmo que o usuário nunca ouça-o (não recomendado);
  • metadata, que carrega apenas algumas informações (mais leve), permitindo assim, caso necessário, uma execução mais rápida;
  • none, que não carrega nada do audio até que o usuário execute-o. Esta é uma estratégia mais efeciente quando a ideia é poupar banda do cliente, porem, ocasiona uma maior espera do cliente até que o aúdio seja executado.

Além deste atributo, devemos adiconar também o constrols, que permite visualizar um player e o loop, que reinicia o aúdio após seu término.

Importante ressaltar que, é interessante adicionar vários <source>'s, como diferentes formatos de aúdio, uma vez que, o navegador que a pessoa está utilizando pode não ter suporte aquele tipo de arquivo. Alem disso, é bacana deixar um <p>, dentro do <audio>, com o link do dowload do som, caso o navegador não tenha suporte a nenhum dos formatos.
