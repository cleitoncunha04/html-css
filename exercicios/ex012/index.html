Vídeos em HTML5

Adicionando vídeos localmente:

É possível adicionar vídeos no servidor local do site. Para isso, devemos ter o arquivo baixado, com uma extensão compatível ao navegador (.mp4/.m4v, .webm ou .ogv). Após isso, devemos utilizar a tag <video> e, assim como nas imagens dinâmicas, utilizar a tag <source>. É interessante ter vários <source>'s com o mesmo arquivo em diferentes extensões, a fim de adaptar o site a diferentes navegadores.

Obs: não é recomendado utilizar vídeos locais, uma vez que, consome muito da hospedagem, encarecendo o serviço de hospedagem.

Adicionando vídeos externamente:

É possível adicionar vídeos salvos em servidores externos, reduzindo o consumo de hospedagem do site. Como possíveis soluções temos o Youtube e Vimeo. Para isso, basta selecionar o vídeo, clicar em compartilhar e ir na opção incoporar. Após isso, só copiar os comandos HTML e colar em seu site.
