Outras formatações

Código-Fonte com Pré-formatação

Para deixar o texto monoespaçado, isto é, todas os caracteres ocupando o mesmo espaço, basta utilizar a tag <code>: var a = 2 + 4.0

Importante ressaltar, que o <code> não mantem a identação do código. Para isso, é necessário utilizar a tag <pre>: 

    
        bool verificarPar(int num)
        {
            bool isPar = false;
            if(num % 2 == 0)
            {
                isPar = true;
            }

            return isPar;
        }

Obs: o <pre> mantem TODOS os espaçamentos e quebras de linha dentro do arquivo HTML.
