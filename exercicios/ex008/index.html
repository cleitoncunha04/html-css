Pricipais formatações

Negrito/Destaque

Nesta frase, temos um termo em negrito, usando a tag <b> (não semântica)

Nesta frase, temos um termo em destaque, usando a tag <strong> (semântica)

Itálico/Ênfase

Nesta frase, temos um termo em itálico, usando a tag <i> (não semântica)

Nesta frase, temos um termo em ênfase, usando a tag <em> (semântica)

Efeito de marca texto

Podemos criar, também, um efeito de marca texto, com a tag <mark> (semântico, com CSS exclusivo para ESTA tag)

É possível criar este efeito com CSS geral (cabeçalho da página), como visto aqui!!!

Texto grande e pequeno

É possível aumentar o tamanho de um texto com a tag <big>, como visto aqui. Também é possível reduzí-lo, com o <small>. Vale lembrar que o <big> é obsoleto, diferente do <small>.

Texto deletado e inserido

Também é possível deletar um texto, com a tag <del> (semântica, ao contrário do <stroke>), visível aqui.
Em contrapartida, ao <del>, temos o <ins>, que "insere" o texto (deixa sublinhado, sendo semântico)

Texto subscrito e sobescrito

Podemos também, subscrever um texto (deixá-lo no formato de molécula), com a tag <sub>: H20.
Já, caso queira sobrescrever um texto (potenciação, por exemplo), podemos usar a tag <sup>:2(2+x) = 5
